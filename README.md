# Task_manager
A simple command-line task manager written in Python.

Features:
- Add/complete tasks
- Persistent storage using JSON
- Colored CLI output
- Loading times for a smoother experience
- First-time instructions

How do I use it?
- Run it with Bash by entering "python Task_manager.py". Afterwards, run it ever so often to manage your personal tasks!

Why did you build this?
- In order to practice fundamental Python & build a working first application.
